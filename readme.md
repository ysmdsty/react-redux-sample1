## this is todo sample app(react redux).

```
yarn add @babel/polyfill
yarn add express prop-types
yarn add --dev @babel/core @babel/cli @babel/preset-env @babel/preset-react
yarn add --dev babelify browserify react react-dom react-redux redux
yarn add --dev webpack webpack-cli webpack-dev-server babel-loader

## express プロジェクト生成
$ npm install express-generator -g
$ express myapp

## node hot reload
$ npm install nodemon -g
$ nodemon ./bin/www
```
